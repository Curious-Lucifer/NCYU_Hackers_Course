## 開啟 Lab
如果是第一次開啟 Lab，請執行以下指令：
```bash
cd PPC_Lab
./snippet build
./snippet start
```

```bash
cd Crypto_Lab
./snippet build
./snippet start
```

如果不是第一次開啟 Lab，請執行以下指令：
```bash
cd PPC_Lab
./snippet start
```

```bash
cd Crypto_Lab
./snippet start
```

---
## 關閉 Lab
```bash
cd PPC_Lab
./snippet stop
```

```bash
cd Crypto_Lab
./snippet stop
```
